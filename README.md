# NCC Albania — Website

Official website of **NCC Albania** (National Competence Centre for High-Performance Computing),
part of the **EuroCC 3** project within the **EuroHPC Joint Undertaking**.

- Bilingual (Albanian / English)
- Led by **AKSHI**, with the **University of Tirana** as academic partner
- Static site (HTML/CSS/JS) — hosted on GitHub Pages

## Structure
- `index.html` — main page
- `privacy.html` — privacy policy (GDPR / Law no. 124/2024)
- `assets/` — logos and branding

## Funding
EuroCC 3 has received funding from the European High-Performance Computing Joint Undertaking (JU)
under Grant Agreement No. 101306701.
